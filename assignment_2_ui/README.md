### Assignment 2 UI

### Installation And Run
1. Clone the repository and navigate to the folder:
```bash
   $ cd assignment_2_ui
``` 
2. Install dependencies:
```bash
   $ npm install
```
3.RunServer
```bash
   $ npm run dev
```
The server will start on http://localhost:3000
